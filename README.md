# Microsoft-PowerBI-Dataviz-World-Champs-2026

## Overview

This project was created as a submission for the Microsoft Power BI Dataviz World Champs 2026.

The challenge focused on analyzing revenue performance and identifying the key drivers behind business growth using Power BI.

The objective was to answer:

"What is driving changes in revenue performance over time, and where should the business focus next to sustain growth?"

The dashboard was designed for executive leadership teams and combines revenue analysis, customer insights, product performance, regional trends, channel performance, and strategic recommendations into a single interactive reporting experience.
